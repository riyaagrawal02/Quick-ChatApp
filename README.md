# QuickChat – Real-Time Chat Application

QuickChat is a lightweight and fast real-time messaging application built with a modern tech stack. It provides seamless communication, secure authentication, and a clean user experience. Designed for personal conversations, team messaging, and small communities.

---

## 🚀 Features

- ⚡ Real-time messaging using WebSocket 
- 🔐 Secure authentication with JWT + bcrypt  
- 🟢 Online / Offline user status  
- 💬 Clean and modern UI with React + Tailwind  
- 📚 Chat history saved in MongoDB  
- 📱 Fully responsive layout  
- 🛡️ Protected routes & error handling  
- 🌐 Scalable backend architecture  

---

## 🧰 Tech Stack

**Frontend:** React, Vite, Tailwind CSS  
**Backend:** Node.js, Express.js  
**Real-Time:** Socket.io  
**Database:** MongoDB + Mongoose  
**Auth:** JWT, bcrypt  
**Deployment:** Vercel
